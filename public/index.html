VibeServer

VibeServer est un serveur backend qui fournit une API pour la manipulation de fichiers et projets, conçu pour être utilisé avec VibePMCP et Claude Desktop.

Statut du Serveur

Le serveur VibeServer est actuellement en cours de vérification...

Points d'accès API

Projets

GET /api/projects/list

Liste tous les projets disponibles

POST /api/projects/create

Crée un nouveau projet

DELETE /api/projects/delete

Supprime un projet existant

Fichiers

GET /api/files/list

Liste les fichiers d'un répertoire

GET /api/files/read

Lit le contenu d'un fichier

POST /api/files/write

Écrit dans un fichier

PATCH /api/files/edit-lines

Modifie des lignes spécifiques d'un fichier

DELETE /api/files/delete

Supprime un fichier

MCP (Model Context Protocol)

POST /api/mcp/execute

Exécute une commande MCP

GET /api/mcp/context

Récupère le contexte MCP actuel
